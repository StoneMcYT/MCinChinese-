# MCinManderin-
A simple ressocepack that overrides the current MAC language stuff and replaces it in the format of.

Characters/pinyin/word

A few examples of changes are:
- 空气 / Kōngqì / Air
- 障碍/ Zhàng'ài /Barrier


<h2> Why this RessocePack? </h2>
Well I'm a programmer and a student so i was thinking of writing a RessocePack that may help other students learn the language in a fun way.
